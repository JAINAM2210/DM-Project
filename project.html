MusiOtic - A DNA Data Storage Approach

Working of MusiOtic

For storing digital data in the form of DNA, we first need to convert binary codes into DNA codes using an algorithm. Our project MusiOtic works in this step of the whole DNA Data Storage process.

Our approach is to take digital data, such as audios and images in their respective file formats, and convert them into DNA files that can be used in the process of DNA synthesis. Currently, MusiOtic is capable of converting audio files into DNA files. Let's take a look at how this conversion is done in MusiOtic.

Converting Audio/Music Files into DNA Files

The main function of MusiOtic is to convert digital audio files into DNA files. We know how these digital audio files are stored in computer memory in binary representation. Therefore, we use this binary file in DNA conversion.

The first step in converting an audio file (.mp3) into a DNA file is to retrieve the binary file of the given audio file. The .mp3 file is first converted into a .bin file, which represents the binary representation of the .mp3 digital audio file. Then, using our algorithm, we convert this binary file into a DNA file (i.e., .bin to .dna).

Algorithm Used in Converting .bin to .dna

We know that a DNA sequence consists of 4 bases (A, T, G, and C). In order to represent the binary file as a DNA file, we need to map these 4 bases to specific binary patterns. With this idea, we came up with the following mapping scheme:

A pattern of '00' represents the first base 'A', '01' represents 'T', '10' represents 'G', and '11' represents 'C'. Using this mapping scheme, we convert the binary file (.bin) into a DNA file (.dna).

Role of RSA Encryption Algorithm in MusiOtic

Nowadays, Data Security is an important aspect of Data Storage. Data Security prevents the crucial data from being stolen, ending up in wrong hands, data breach, etc. It also protects the data from hacking. If our data is encrypted and it gets hacked, then also our data is safer. So, encryption of data is very important for the security of our data.

We have developed MusiOtic keeping in mind the Data Safety of the user. In order to encrypt the users' data while creating the DNA Files, we have used RSA Encryption Algorithm. So, the .mp3 Digital Audio File given by the user is first converted into an encrypted .bin file using RSA Algorithm. After this, the encrypted .bin file is converted into .dna file, and later this encrypted .dna file is used in storing data in DNA sequence. Therefore, the data stored in the DNA sequence is not directly accessible, which makes it safe.

While converting from .dna file back to .mp3 file, the encrypted .dna file is first converted to an encrypted .bin file. This encrypted .bin file is then decrypted using the decryption algorithm, and it is converted back to the original .mp3 file. Thus, storing an encrypted DNA sequence protects the data in various aspects.
